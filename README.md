# Yet another QR-code business card generator


Use a minimal JSON Schema for vcard 2.1 to generate a QR code that can be scanned by any modern smartphone. The QR code contains all the information needed to add a contact to the address book. This simple python script uses qrcode and PIL to generate the QR code image and reportlab for pdf generation.

Included example:

![Forrest's card](img/Example.png)
